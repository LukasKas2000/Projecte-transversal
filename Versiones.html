Versiones

Aquí podras ver la infinidad de versiones por las que ha pasado el juego hasta el dia de hoy.


  • Versión 1.20 - Fecha de lanzamiento: Junio 2023 - Novedades: nuevos biomas, nuevos mobs.

  • Versión 1.19 - Fecha de lanzamiento: Junio 2022 - Novedades: el Warden, bioma Oscuridad Profunda.

  • Versión 1.18 - Fecha de lanzamiento: Noviembre 2021 - Novedades: actualización de montañas y cuevas.

  • Versión 1.17 - Fecha de lanzamiento: Junio 2021 - Novedades: nuevas cuevas, ametistas, nuevos bloques de minerales.

  • Versión 1.16 - Fecha de lanzamiento: Junio 2020 - Novedades: actualización del Nether, nuevos biomas y mobs.

  • Versión 1.15 - Fecha de lanzamiento: Diciembre 2019 - Novedades: abejas, colmenas, bloques de miel.

  • Versión 1.14 - Fecha de lanzamiento: Abril 2019 - Novedades: actualización de aldeas, nuevos bloques.

  • Versión 1.13 - Fecha de lanzamiento: Julio 2018 - Novedades: actualización acuática, nuevos mobs marinos.

  • Versión 1.12 - Fecha de lanzamiento: Junio 2017 - Novedades: bloques de terracota, nuevos colores.

  • Versión 1.11 - Fecha de lanzamiento: Noviembre 2016 - Novedades: Mansiones del bosque, Vindicator, Evoker.

  • Versión 1.10 - Fecha de lanzamiento: Junio 2016 - Novedades: osos polares, nuevos bloques, iglús.

  • Versión 1.9 - Fecha de lanzamiento: Febrero 2016 - Novedades: actualización de combate, escudos, nuevos ítems.

  • Versión 1.8 - Fecha de lanzamiento: Septiembre 2014 - Novedades: nuevos bloques, Guardians, nuevos biomas.

  • Versión 1.7 - Fecha de lanzamiento: Octubre 2013 - Novedades: nuevos biomas, nuevos árboles, nuevas flores.

  • Versión 1.6 - Fecha de lanzamiento: Julio 2013 - Novedades: caballos, armaduras para caballos, heno.

  • Versión 1.5 - Fecha de lanzamiento: Marzo 2013 - Novedades: actualización de redstone, tolvas, bloques de redstone.

  • Versión 1.4 - Fecha de lanzamiento: Octubre 2012 - Novedades: Wither, calabazas talladas, lobos entrenados.

  • Versión 1.3 - Fecha de lanzamiento: Agosto 2012 - Novedades: esmeraldas, comercio con aldeanos, templos.

  • Versión 1.2 - Fecha de lanzamiento: Marzo 2012 - Novedades: junglas, golems de hierro, nuevos bloques.

  • Versión 1.1 - Fecha de lanzamiento: Enero 2012 - Novedades: nuevos idiomas, huevos de spawn, ajustes de juego.

  • Versión 1.0 - Fecha de lanzamiento: Noviembre 2011 - Novedades: lanzamiento oficial del juego, el Nether, el End y el dragón.

  • Beta 1.8 - Fecha de lanzamiento: Septiembre 2011 - Novedades: nuevo sistema de experiencia, Enderman, pueblos.

  • Beta 1.7 - Fecha de lanzamiento: Junio 2011 - Novedades: pistones, cizallas.

  • Beta 1.6 - Fecha de lanzamiento: Mayo 2011 - Novedades: mapas, nuevos bugs.

  • Beta 1.5 - Fecha de lanzamiento: Abril 2011 - Novedades: clima (lluvia, nieve, tormentas), rieles de detector.

  • Beta 1.4 - Fecha de lanzamiento: Marzo 2011 - Novedades: lobos, camas.

  • Beta 1.3 - Fecha de lanzamiento: Febrero 2011 - Novedades: repetidores de redstone, cofres dobles.

  • Beta 1.2 - Fecha de lanzamiento: Enero 2011 - Novedades: lapislázuli, bloques de nota.

  • Beta 1.1 - Fecha de lanzamiento: Enero 2011 - Novedades: modificaciones menores y correcciones de bugs.

  • Beta 1.0 - Fecha de lanzamiento: Diciembre 2010 - Primera versión oficial en fase beta.

  • Alpha 1.2.6 - Fecha de lanzamiento: Diciembre 2010 - Novedades: redstone, musgo, cañas de pescar.

  • Alpha 1.2 - Fecha de lanzamiento: Octubre 2010 - Novedades: calabazas, relojes, pescado.

  • Alpha 1.1 - Fecha de lanzamiento: Septiembre 2010 - Novedades: nuevas herramientas, armaduras, mobs hostiles.

  • Alpha 1.0 - Fecha de lanzamiento: Junio 2010 - Primer lanzamiento de la versión Alpha.

  • Infdev - Fecha de lanzamiento: Febrero 2010 - Introducción del modo supervivencia.

  • Classic - Fecha de lanzamiento: Mayo 2009 - Versión inicial, solo modo creativo.
